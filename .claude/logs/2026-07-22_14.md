- `src/services/sleeper/league-matchups.ts` — syncLeagueMatchups(): league status fetch (finality) → per-week fetch/validate sweep → matchups rows (matchup_period=week, is_home null, points/custom_points as-received, fetched_at/is_final stamped) + player_scores rows (points as-received from players_points, was_starter from that week's starters set, roster attribution) → missing-player skip+record → chunked upserts (500). effective_points is a stored generated column — excluded from inserts (generated types list it insertable; Postgres would reject).
