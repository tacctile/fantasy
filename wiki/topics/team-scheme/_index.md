# Team Scheme

Team-level context that determines player opportunity and value.

| Page | Description | Confidence |
| ---- | ------------ | ---------- |
| [proe](proe.md) | Actual pass rate minus a model's expected pass rate given game situation — isolates scheme-driven passing tendency from game-script variance; scramble classification is the main provider disagreement. | medium |
| [neutral-game-script-pass-rate](neutral-game-script-pass-rate.md) | Pass rate filtered to close-score situations, removing garbage-time inflation and clock-killing depression; the score-margin/time-window threshold is contested and unstandardized. | medium |
| [pace-of-play](pace-of-play.md) | Plays per game as a volume multiplier on all opportunity metrics; raw play count is confounded by efficiency, opponent pace, and game script — distinct from tempo (seconds per snap). | medium |
| [offensive-line-continuity](offensive-line-continuity.md) | Stability of the same five starters across games; sources genuinely disagree on how much independent predictive power it carries once team quality/selection bias is controlled for — treated as an open dispute, not settled. | medium |
| [offensive-line-blocking-efficiency](offensive-line-blocking-efficiency.md) | Pass-block and run-block efficiency metrics (charting/grading, timing-based win-rate, and outcome-adjusted families); confounded by QB pocket presence and RB vision respectively — no universal composite exists. | medium |
| [play-action-rate](play-action-rate.md) | Share of pass plays with a run fake; reliably boosts QB efficiency by freezing second-level defenders, but RPO classification is the dominant unresolved cross-provider disagreement — usage and efficiency must be evaluated separately. | medium |
