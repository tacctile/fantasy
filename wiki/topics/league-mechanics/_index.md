# League Mechanics

Scoring formats, draft strategy, and league structure.

| Page | Description | Confidence |
| ---- | ------------ | ---------- |
| [ppr-half-ppr-standard-scoring](ppr-half-ppr-standard-scoring.md) | Reception scoring restructures relative positional value through a replacement-level shift, not raw point addition; value must be measured as points-above-replacement, and target share/route participation predict PPR value better than yards-per-touch or touchdown rate. | high |
| [te-premium-scoring](te-premium-scoring.md) | TE Premium is a scarcity amplifier that benefits route-running, target-earning tight ends specifically — not the position broadly — and widens rather than closes the gap between the elite tight end tier and replacement-level options. | high |
| [superflex-two-qb-value-shift](superflex-two-qb-value-shift.md) | Superflex/2QB formats collapse the quarterback replacement-level baseline nonlinearly due to a hard ~32-starter NFL supply cap; 2QB (mandatory) is stricter than Superflex (optional), and rushing production is disproportionately valuable under common scoring. | high |
